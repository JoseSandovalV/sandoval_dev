Proyecto Integrador: PLH - Sistema de Gestión de Becas Alimentarias

Publicado el 12 de Mayo de 2025 por Jose Sandoval

Descripción del proyecto

Este proyecto surge como una respuesta a la necesidad de optimizar el proceso de gestión de becas alimentarias en instituciones educativas. Desarrollado con Laravel y MySQL, el sistema permite realizar solicitudes, administrar roles y dar seguimiento a los beneficiarios. Fue presentado en el concurso InnovaTEC 2025, representando al equipo de Ingeniería Informática del Instituto Tecnológico Superior de Lerdo.

Tecnologías utilizadas

Laravel MySQL Bootstrap GitHub Breeze

Dificultades y lecciones aprendidas

  • Comprensión inicial de los componentes de Laravel y su estructura MVC.
  • Gestión de roles y permisos en un entorno con múltiples niveles de usuario.
  • Implementación de autenticación segura usando Laravel Breeze.
  • Trabajo colaborativo utilizando Git y GitHub de manera eficiente.
  • Aprendizaje constante frente a problemas inesperados durante el desarrollo.
